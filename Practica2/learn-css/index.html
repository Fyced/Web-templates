Learn HTML and CSS using a free website template

This website template was created for a step-by-step tutorial called "Building your first website using a free website template", written by professional website template designer Andreas Viklund. The main purpose of the tutorial is to provide an example of how a free website template can be used to build a functional, accessible and technically good website, even with no prior experience of HTML and CSS. By following the tutorial, the reader will get familiar with the concept of working with code using small steps. Those who want to learn more can do so by diving deeper into the CSS to modify the layout and the design, while those who only want to build a website can focus on editing the HTML and the content.

A dynamic template for a dynamic tutorial

This template is used as the learning example for the tutorial, and the first version of it was released when part 2 of the tutorial was published. As future parts are published, new versions of this template will be published as well. The development of the template (as well as of the tutorial) is based on feedback from the readers, so there may be additions and modifications to future versions of this template based on suggestions from anyone who is following the tutorial. If you want to learn about something specific, go to the first part of the tutorial and post a comment with your feedback.

The template can of course also be used as any other template from andreasviklund.com. It is free to use for both personal or commercial use, and free to modify in any way anyone may want to.
